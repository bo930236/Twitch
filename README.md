# Twitch API

[![Live Demo](https://img.shields.io/badge/demo-online-green.svg)](https://twitch-git-master-bo930236.vercel.app" "前往 DEMO")

## Features

- 瀏覽 Twitch 最熱門的五種遊戲直播
- 頁面往下會持續提供直播頻道資訊
- 點擊縮圖可以前往 Twitch 收看頻道
